put /activate-enlace/:id
Activa un enlace existente (activateEnlace)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body Enlace (required)
Body Parameter — Se necesita el ID del enlace a activar o desactivar

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

message: Enlace activado/desactivado

404

message: No se ha encontrado ningún enlace

500

message: Error del servidor

503

message: La petición no tiene cabecera de autenticación
